### Background:
#### Grocery List
This script takes in a list of items and outputs the count of each item.

### Getting Started:
Export this directory using SVN.
```
svn export https://github.com/GrandEchoWhiskey/harvard-cs50-python-projects/trunk/pset-3-grocery
```
Change directory
```
cd pset-3-grocery
```
Now run the script
```
python grocery.py
```
