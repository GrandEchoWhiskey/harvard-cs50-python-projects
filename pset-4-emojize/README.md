### Background:
#### Emoji Converter
This Python script converts text input to emojis using the emoji library.

### Getting Started:
Export this directory using SVN.
```
svn export https://github.com/GrandEchoWhiskey/harvard-cs50-python-projects/trunk/pset-4-emojize
```
Change directory
```
cd pset-4-emojize
```
Now run the script
```
python emojize.py
```
